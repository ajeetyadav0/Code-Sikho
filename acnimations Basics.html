CSS Animations Basics

What are CSS Animations?

CSS animations allow you to transition between CSS styles over time, creating motion and visual effects without JavaScript.

@keyframes animationName { from { /* Start styles */ } to { /* End styles */ } } .element { animation: animationName 2s infinite; }

Animation Basics

Fade In Animation

Fade In
@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } .fade-in { animation: fadeIn 2s infinite alternate; }

Slide In Animation

Slide In
@keyframes slideIn { from { transform: translateX(-100%); } to { transform: translateX(0); } } .slide-in { animation: slideIn 2s infinite alternate; }

Animation Properties

Animation Shorthand

.element { animation: name duration timing-function delay iteration-count direction fill-mode; } /* Example */ .element { animation: slideIn 2s ease-in-out 1s infinite alternate forwards; }

Individual Properties

.element { animation-name: slideIn; animation-duration: 2s; animation-timing-function: ease-in-out; animation-delay: 1s; animation-iteration-count: infinite; animation-direction: alternate; animation-fill-mode: forwards; }

Timing Functions

/* Common timing functions */ animation-timing-function: ease; /* Default */ animation-timing-function: ease-in; animation-timing-function: ease-out; animation-timing-function: ease-in-out; animation-timing-function: linear; animation-timing-function: cubic-bezier(0.1, 0.7, 1.0, 0.1);

Animation Examples

Pulse Animation

Pulse
@keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .pulse { animation: pulse 2s infinite; }

Rotate Animation

Rotate
@keyframes rotate { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .rotate { animation: rotate 3s linear infinite; }

Bounce Animation

Bounce
@keyframes bounce { 0%, 20%, 50%, 80%, 100% { transform: translateY(0); } 40% { transform: translateY(-30px); } 60% { transform: translateY(-15px); } } .bounce { animation: bounce 2s infinite; }

Color Change Animation

Color
@keyframes colorChange { 0% { background-color: #4CAF50; } 25% { background-color: #2196F3; } 50% { background-color: #FF9800; } 75% { background-color: #9C27B0; } 100% { background-color: #4CAF50; } } .color-change { animation: colorChange 5s infinite; }
