CSS clamp() Function Magic

Responsive Font Size

font-size: clamp(1rem, 4vw, 2.5rem);

This text scales between 1rem and 2.5rem based on viewport width!

The clamp() function takes three values: minimum, preferred, and maximum. The font size will be 4vw of the viewport width, but won't go below 1rem or above 2.5rem.

Responsive Width

width: clamp(200px, 50%, 800px);
Responsive Width Box

This box has a width that's 50% of its container, but won't be narrower than 200px or wider than 800px.

Responsive Margin

margin: clamp(10px, 5vw, 50px);
Responsive Margin Box

This box has a margin that scales with the viewport width, but stays between 10px and 50px.
