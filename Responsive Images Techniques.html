Responsive Images Techniques

Why Responsive Images Matter

Responsive images ensure optimal loading times and display quality across all devices, improving performance and user experience.

Basic Responsive Image

.responsive-img { max-width: 100%; height: auto; display: block; } <img src="image.jpg" alt="Description" class="responsive-img">

✅ Simple and effective for basic responsiveness

✅ Images scale down but never larger than original

Srcset Attribute

<img src="image-small.jpg" srcset="image-small.jpg 400w, image-medium.jpg 800w, image-large.jpg 1200w" sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px" alt="Description">

✅ Browser chooses optimal image size based on viewport

✅ Improves loading performance

Picture Element for Art Direction

<picture> <source media="(max-width: 600px)" srcset="image-crop-mobile.jpg"> <source media="(max-width: 900px)" srcset="image-crop-tablet.jpg"> <img src="image-desktop.jpg" alt="Description"> </picture>

✅ Different images for different screen sizes

✅ Perfect for art direction and cropping

Modern Image Formats

<picture> <source srcset="image.webp" type="image/webp"> <source srcset="image.jp2" type="image/jp2"> <img src="image.jpg" alt="Description"> </picture>

✅ WebP offers better compression than JPEG

✅ Fallback to JPEG for unsupported browsers

Lazy Loading

<img src="image.jpg" alt="Description" loading="lazy">

✅ Native browser lazy loading

✅ Improves initial page load performance

Responsive Image Gallery

.image-gallery { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); gap: 15px; } .gallery-img { width: 100%; height: 150px; object-fit: cover; }

✅ Responsive grid layout

✅ Images maintain aspect ratio with object-fit

Image Optimization Tips

✅ Do's

  • Use appropriate image formats (WebP, AVIF)
  • Compress images before uploading
  • Use srcset for responsive images
  • Implement lazy loading for below-the-fold images
  • Use the picture element for art direction
  • Include width and height attributes to prevent layout shift

❌ Don'ts

  • Don't use large images for small spaces
  • Don't forget alt attributes for accessibility
  • Don't rely solely on CSS for image resizing
  • Don't use images for text
  • Don't ignore image format support

Complete Responsive Image Example

<picture> <!-- Modern formats --> <source srcset="image.webp" type="image/webp"> <source srcset="image.avif" type="image/avif"> <!-- Responsive sizes --> <source media="(max-width: 600px)" srcset="image-small.webp 400w, image-medium.webp 800w" sizes="100vw"> <source media="(max-width: 900px)" srcset="image-medium.webp 800w, image-large.webp 1200w" sizes="100vw"> <!-- Fallback --> <img src="image-fallback.jpg" srcset="image-small.jpg 400w, image-medium.jpg 800w, image-large.jpg 1200w" sizes="(max-width: 600px) 400px, (max-width: 900px) 800px, 1200px" alt="Description" loading="lazy" width="1200" height="800"> </picture>

This comprehensive example includes modern formats, responsive sizing, art direction, lazy loading, and dimension attributes.
