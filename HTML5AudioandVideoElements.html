HTML5 Audio and Video Elements

Video Element

<video controls poster="image.jpg"> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> </video>
✅ Multiple sources for browser compatibility
✅ Poster image shows before video plays
✅ Controls attribute adds play/pause buttons

Audio Element

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> </audio>
✅ Multiple formats for browser compatibility
✅ Native browser controls for playback
