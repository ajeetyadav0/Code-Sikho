The Secret to Responsive Design

A Practical Guide to Mastering CSS Flexbox

1. The Foundation: `display: flex;`

This is the switch that turns on Flexbox. It turns a direct parent into a flex container and its children into flex items. By default, items will try to fit on one line.

.container {
  display: flex;
}

2. The Main Axis: `flex-direction`

Flexbox is one-dimensional. It works along a main axis (horizontal by default). You can easily switch it to a vertical column.

.container {
  display: flex;
  flex-direction: column; /* or row (default) */
}

3. The Secret Sauce: `flex-wrap`

This is a key to responsiveness! By default, items try to squish onto one line. With `flex-wrap: wrap`, items can flow onto the next line when they run out of space.

.container {
  display: flex;
  flex-wrap: wrap; /* or nowrap (default) */
}

4. Alignment & Spacing

Easily control the distribution and alignment of items. `justify-content` works on the main axis, while `align-items` works on the cross axis.

.container {
  display: flex;
  justify-content: space-between; /* center, flex-end, space-around... */
  align-items: center; /* stretch, flex-start, flex-end... */
}

5. The Ultimate Secret: The `flex` Property

This is a shorthand for `flex-grow`, `flex-shrink`,` and `flex-basis`. It tells items how to behave in a flexible container. `flex: 1` is a magic value that tells an item to grow and fill any available space.

.item {
  flex: 1; /* Shorthand for: 1 1 0% */
  /* flex-grow: 1;   (can grow) */
  /* flex-shrink: 1; (can shrink) */
  /* flex-basis: 0%; (start with no size) */
}
