HTML Meta Tags Explained

Basic Meta Tags

<meta charset="UTF-8">

Specifies the character encoding for the HTML document.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Makes the website responsive on mobile devices.

<meta name="description" content="...">

Provides a brief description of the page for search engines.

Open Graph Meta Tags

<meta property="og:title" content="...">

Defines the title when shared on social media platforms like Facebook.

<meta property="og:image" content="...">

Specifies the image that appears when the page is shared.

Twitter Card Meta Tags

<meta name="twitter:card" content="summary_large_image">

Defines the type of Twitter card to display.
