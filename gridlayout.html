CSS Grid Layout Explained

What is CSS Grid?

CSS Grid is a two-dimensional layout system for creating complex web layouts.

.container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: auto 1fr auto; gap: 10px; }
