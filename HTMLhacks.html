2. Input Validation Hacks

Use HTML5 validation attributes to provide instant feedback without JavaScript.

<input type="email" required>
<input type="text" pattern="[A-Za-z]{3}" title="Three letters">
<input type="text" minlength="5" maxlength="10">
<input type="number" min="1" max="100">
<input type="text" placeholder="Enter your name" required>

3. Password Strength Indicator

Create a visual password strength indicator with just HTML and CSS.

8. Input Mode and Enter Key Hacks

Control the virtual keyboard and form submission behavior.

<input type="text" inputmode="numeric" pattern="[0-9]*" placeholder="Numeric keypad">
<input type="text" inputmode="tel" placeholder="Phone keypad">
<input type="text" inputmode="email" placeholder="Email keyboard">
<input type="text" inputmode="url" placeholder="URL keyboard">
<form>
  <input type="text" formaction="/search" placeholder="Search (Enter to submit)">
  <button type="submit" style="display:none">Submit</button>
</form>

9. Multiple Selection Hacks

Enhance checkbox and radio button selections.

<input type="checkbox" id="selectAll">
<label for="selectAll">Select All</label>

<input type="checkbox" name="option" value="1" class="selectable">
<label for="option1">Option 1</label>
<input type="checkbox" name="option" value="2" class="selectable">
<label for="option2">Option 2</label>
<input type="checkbox" name="option" value="3" class="selectable">
<label for="option3">Option 3</label>
