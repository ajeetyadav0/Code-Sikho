Dark Mode Toggle

About This Demo

This is a demonstration of a dark mode toggle using CSS variables and JavaScript. Click the toggle in the top-right corner to switch between light and dark themes.

The page's colors are defined in CSS variables, which are then updated based on the selected theme. This approach provides a smooth transition between themes and is easy to maintain.

How It Works

The toggle switch sets a `data-theme` attribute on the `body` element. CSS then uses attribute selectors to apply different variable values for light and dark themes.

The theme preference is also saved to `localStorage` so it persists across page reloads.
