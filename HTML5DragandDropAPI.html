HTML5 Drag and Drop API

What is Drag and Drop?

HTML5 Drag and Drop API allows users to drag and drop elements on a webpage.

// Make an element draggable <div draggable="true">Drag me</div> // Handle drag events element.addEventListener('dragstart', function(event) { // Code to handle drag start }); element.addEventListener('drop', function(event) { // Code to handle drop });
