CSS :is() Selector Magic

What is the :is() Selector?

The :is() pseudo-class function takes a selector list as its argument, and selects any element that can be selected by one of the selectors in that list. This is useful for writing large, complex selectors in a more compact form.

Before :is(), you would have to write each selector separately, which could lead to long and repetitive CSS rules.

Example: Styling Headings and Paragraphs

Without :is()

.old-way h1, .old-way h2, .old-way h3 { color: #ff6b6b; }

.old-way p, .old-way li { color: #4ecdc4; }

With :is()

.new-way :is(h1, h2, h3) { color: #ff6b6b; }

.new-way :is(p, li) { color: #4ecdc4; }

Example: Navigation Styling

.nav :is(a, button) { display: block; padding: 10px; }

.nav :is(a, button):hover { background-color: #e9ecef; }

.nav :is(a, button):active { background-color: #dee2e6; }

Example: Form Styling

.form-example :is(input[type="text"], input[type="email"], textarea) { width: 100%; padding: 10px; }

.form-example :is(input[type="text"]:focus, input[type="email"]:focus, textarea:focus) { border-color: #6c5ce7; }

Example: Combining with :not()

.list-example :is(li:not(.first), li:not(.last)) { border-bottom: 1px solid #eee; }

Browser Support

The :is() pseudo-class is supported in all modern browsers, including Chrome, Firefox, Safari, and Edge. It's a great tool for writing more maintainable and readable CSS.
