Parallax Scrolling

Create depth and visual interest with multi-layer scrolling effects

How Parallax Works

Parallax scrolling creates an illusion of depth by moving background layers at different speeds than the foreground. As you scroll, the elements appear to move at different rates, creating a 3D effect.

Background Layer

The background layer moves at the slowest speed, creating a sense of distance.

Middle Layer

The middle layer moves at a medium speed, creating depth between foreground and background.

Foreground Layer

The foreground layer moves at the fastest speed, appearing closest to the viewer.

// Parallax Scrolling Effect
window.addEventListener('scroll', () => {
    const scrolled = window.pageYOffset;
    
    // Get all parallax elements
    const parallaxElements = document.querySelectorAll('[data-speed]');
    
    parallaxElements.forEach(element => {
        const speed = element.dataset.speed || 0.5;
        const yPos = -(scrolled * speed);
        
        element.style.transform = `translateY(${yPos}px)`;
    });
});

// HTML Structure
<div class="parallax-container">
    <div class="parallax-layer bg-layer"></div>
    <div class="parallax-layer mountain-layer" data-speed="0.5"></div>
    <div class="parallax-layer cloud-layer" data-speed="0.2"></div>
    <div class="parallax-layer content-layer">
        <h1>Parallax Scrolling</h1>
        <p>Content here</p>
    </div>
</div>
