Image Comparison Slider

Drag the slider to compare the before and after images. This is perfect for showing transformations, renovations, or any visual changes.

You can customize the slider by changing the colors, labels, and even the orientation (horizontal or vertical).

To create a vertical slider, change the width and height properties of the slider, and adjust the cursor value to ns-resize.

For a more advanced effect, you can create a circular slider using CSS conic-gradient and transform properties.
