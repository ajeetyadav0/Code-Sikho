CSS Responsive Design Basics

What is Responsive Design?

Responsive design ensures your website looks and works well on all devices and screen sizes.

/* Basic Media Query */ @media (max-width: 768px) { /* Styles for screens smaller than 768px */ }

Viewport Meta Tag

Essential Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This meta tag tells the browser to set the viewport width to the device's width and set the initial zoom level to 100%.

Responsive Typography

This text will adjust its size based on the viewport width. On mobile, it's 16px, on tablet 18px, and on desktop 20px.

.responsive-text { font-size: 16px; line-height: 1.5; } @media (min-width: 768px) { .responsive-text { font-size: 18px; } } @media (min-width: 1024px) { .responsive-text { font-size: 20px; } }

Responsive Images

.responsive-image { max-width: 100%; height: auto; display: block; }

Setting max-width to 100% ensures images scale down but never larger than their original size.

Mobile First vs Desktop First

Mobile First Approach

Mobile First Content

Start with mobile styles, then enhance for larger screens.

/* Mobile First */ .mobile-first { padding: 15px; font-size: 14px; } @media (min-width: 768px) { .mobile-first { padding: 20px; font-size: 16px; } } @media (min-width: 1024px) { .mobile-first { padding: 30px; font-size: 18px; } }

Desktop First Approach

Desktop First Content

Start with desktop styles, then adjust for smaller screens.

/* Desktop First */ .desktop-first { padding: 30px; font-size: 18px; } @media (max-width: 1023px) { .desktop-first { padding: 20px; font-size: 16px; } } @media (max-width: 767px) { .desktop-first { padding: 15px; font-size: 14px; } }

Common Breakpoints

/* Common Breakpoints */ /* Small devices (phones) */ @media (max-width: 576px) { } /* Medium devices (tablets) */ @media (max-width: 768px) { } /* Large devices (desktops) */ @media (max-width: 992px) { } /* Extra large devices (large desktops) */ @media (max-width: 1200px) { }

These are common breakpoints, but you should adjust them based on your content and design needs.
