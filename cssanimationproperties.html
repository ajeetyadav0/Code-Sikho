CSS Animation Properties

Animation Properties

CSS provides various properties to control animations: duration, timing function, delay, iteration count, direction, fill mode, and play state.

.element { animation-name: move; animation-duration: 2s; animation-timing-function: ease; animation-delay: 0s; animation-iteration-count: 1; animation-direction: normal; animation-fill-mode: none; animation-play-state: running; }
