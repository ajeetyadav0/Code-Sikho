CSS Transitions vs Animations

Key Differences

Transitions: Require a trigger (like hover) and go from start to end state.

Animations: Can run automatically, have multiple keyframes, and more control.

Transition Syntax

.element { transition: property duration timing-function delay; } .element:hover { /* End state */ }

Animation Syntax

@keyframes animationName { 0% { /* Start state */ } 50% { /* Middle state */ } 100% { /* End state */ } } .element { animation: animationName duration timing-function delay iteration-count direction fill-mode; }

Transition Examples

Color Transition

Hover Me
.transition-color { transition: background-color 0.5s ease; } .transition-color:hover { background-color: #2196F3; }

Transform Transition

Hover Me
.transition-transform { transition: transform 0.5s ease; } .transition-transform:hover { transform: scale(1.2) rotate(10deg); }

Multiple Property Transition

Hover Me
.transition-multiple { transition: background-color 0.5s ease, transform 0.5s ease, border-radius 0.5s ease; } .transition-multiple:hover { background-color: #FF9800; transform: translateY(-10px); border-radius: 50%; }

Animation Examples

Auto-playing Pulse Animation

Pulse
@keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.1); } } .animation-pulse { animation: pulse 2s infinite; }

Color Cycle Animation

Color Cycle
@keyframes colorCycle { 0% { background-color: #4CAF50; } 33% { background-color: #2196F3; } 66% { background-color: #FF9800; } 100% { background-color: #4CAF50; } } .animation-color { animation: colorCycle 3s infinite; }

Bounce Animation

Bounce
@keyframes bounce { 0%, 20%, 50%, 80%, 100% { transform: translateY(0); } 40% { transform: translateY(-20px); } 60% { transform: translateY(-10px); } } .animation-bounce { animation: bounce 2s infinite; }

JavaScript-triggered Animation

Click Me
@keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.1); } } .animation-trigger { animation: none; } .animation-trigger.active { animation: pulse 1s 3; } // JavaScript document.getElementById('trigger-animation').addEventListener('click', function() { this.classList.add('active'); setTimeout(() => { this.classList.remove('active'); }, 3000); });
