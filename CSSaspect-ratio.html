What is aspect-ratio?

The aspect-ratio property allows you to set the preferred aspect ratio of an element, ensuring it maintains consistent proportions regardless of the viewport size.

Syntax:

aspect-ratio: width / height;

Common Values:

  • 16 / 9 - Widescreen video
  • 4 / 3 - Traditional photography
  • 3 / 2 - Classic photography
  • 1 / 1 - Square images
  • 21 / 9 - Cinema screen

Benefits:

  • Maintains consistent proportions across different screen sizes
  • Prevents layout shifts when loading content
  • Reduces need for placeholder images
  • Improves page load performance

Browser Support:

Fallback for Older Browsers:

For browsers that don't support aspect-ratio, you can use the padding-top technique:

padding-top: (height / width * 100)%;
